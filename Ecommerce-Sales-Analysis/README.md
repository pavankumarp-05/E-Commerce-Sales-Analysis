# E-Commerce Sales Analysis Dashboard

## Project Overview

This project analyzes Superstore sales data using Python and Power BI. The dataset was cleaned using Python (Pandas), and an interactive dashboard was created in Power BI to analyze sales performance.

## Tools Used

* Python (Pandas)
* Power BI
* Excel

## Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Top Products Analysis
* Region and Category Filters

## Key Insights

* Total Sales: 2.33M
* Total Profit: 292.30K
* Total Orders: 5111
